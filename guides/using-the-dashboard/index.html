Using the Fiona Dashboard

The Fiona Dashboard is a web-based control centre that lets you manage your computer through Fiona — all from your browser. Think of it as a remote control for your desktop, but it runs on your own machine.

You don't need to know how to code to use it. If you can open a web page and click buttons, you're good.

Starting the Dashboard

You only need to do this once per session.

The project includes two handy scripts that handle everything for you:

Option 1: Using the startup script (recommended)

Open a terminal in the Fiona project folder and run:

./scripts/fiona-pages-start

This automatically starts the server and shows you the address to open.

Option 2: Starting it directly

python3 fionaLocalPages/server/app.py

Opening the Dashboard

Once the server is running, open your web browser and go to:

http://fiona.agent:5000

You should see the Fiona Dashboard — a dark-themed interface with a sidebar on the left.

Tip: If fiona.agent doesn't work, try http://127.0.0.1:5000 instead.

Navigating Around

The sidebar on the left is your main menu. Click any item to open that section.

Common Tasks

Chat with the AI

  1. Click AI Chat in the sidebar.
  2. Type your message in the box at the bottom.
  3. Press Enter or click Send.
  4. Watch the AI respond in real time — words appear as they're generated.

You can change the AI's personality (General, Planner, Engineer, Analyst, Security) using the dropdown.

Browse and Run Actions

  1. Click Actions in the sidebar.
  2. Browse the list or use the search bar.
  3. Each action shows its name, what it does, and how risky it is (safe, medium, or high).
  4. Click Run to execute an action.
  5. If the action needs approval, you'll see a prompt — confirm to proceed.

Use the Terminal

  1. Click Terminal in the sidebar.
  2. Type commands just like you would in a normal terminal.
  3. The terminal tracks which folder you're in (cd actually works).
  4. Press Tab for autocomplete suggestions.
  5. Try help to see available commands, or clear to clear the screen.

Automate a Web Browser

  1. Make sure you have Chrome or Chromium installed on your system.
  2. Click Browser in the sidebar.
  3. Click Start to launch the automated browser (it runs invisibly by default).
  4. Enter a URL and click Navigate.
  5. Use the controls to click elements, type text, or take screenshots.
  6. Click Stop when you're done.

Check Your System Performance

  1. Click Performance in the sidebar.
  2. Watch live-updating charts of your CPU and memory usage.
  3. The charts update automatically every few seconds.

Tips & Tricks

  • Sidebar collapse: Click the collapse button at the bottom of the sidebar to give the main area more room.
  • Dark theme: The dashboard is dark by default (easy on the eyes).
  • Keyboard shortcut: Press Ctrl+K (or Cmd+K on Mac) to open the command palette — a quick-search for pages and actions.
  • Multiple tabs: You can open different sections in different browser tabs — each one works independently.
  • Settings are saved: Your preferences (like sidebar state) are remembered between sessions.

Troubleshooting

Problem Likely Cause What to Try
Dashboard won't load Server not running Run ./scripts/fiona-pages-start from the Fiona folder, or start the server directly with python3 fionaLocalPages/server/app.py
Page shows "Failed to Load" Server restart needed Stop the server (Ctrl+C), start it again, then refresh the page
Terminal commands don't work Command not found The terminal runs on your computer — standard Linux commands are available
Browser controls are greyed out Browser not started Click "Start" first in the Browser page
AI Chat is not responding Ollama not running Make sure Ollama is installed and running (ollama serve in a terminal)
Settings won't save File permissions Check that ~/.config/fiona/settings.txt is writable

Stopping the Dashboard

When you're done, just close the browser tab. To stop the server completely, use the stop script:

./scripts/fiona-pages-stop

Or, if you started it directly, go back to that terminal and press Ctrl+C.

Note: Stopping the server doesn't affect other Fiona features (like keyboard shortcuts or the CLI). Only the web dashboard turns off.
