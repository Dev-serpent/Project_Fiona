Voice Commands

"run macro [name]" Runs a saved macro
Requirements
  • Microphone access enabled in your browser
  • Speech recognition API support (Chrome, Edge, Safari)
  • Backend voice processing service running
How to Use Voice Commands
1 Enable the microphone — Grant microphone permission when prompted by your browser.
2 Activate voice mode — Click the microphone icon in the status bar or press your configured push-to-talk key.
3 Speak a command — Say one of the available voice commands clearly. Fiona will process and execute the action.
4 Watch the feedback — The status indicator will show listening state. A confirmation will appear when a command is recognized.

Voice Commands — This page provides a reference for available voice commands. Full voice interaction requires the backend speech recognition service to be running and connected.
